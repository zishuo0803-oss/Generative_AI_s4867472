# Generative AI Research Project

## Project Overview
This project explores the benefits and risks of using generative AI tools.

## Folder Structure
- Literature_Review: academic sources
- Quantitative_Data: survey data and scripts
- Qualitative_Data: interviews and analysis
- Reports: drafts and final reports
- Visualisations: charts and figures
